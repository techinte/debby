

# debby

to do the scraping, please do the following

Install:

`pip install selenium beautifulsoup4`

(you can also do `pip install -r requirements.txt` instead if you want!)

Please use Firefox browser and install this extension:
https://addons.mozilla.org/en-US/firefox/addon/cookie-editor/

Then, navigate to Twitter and login. (You have to make an account.)
Open the extension by clicking the puzzle piece icon
in your top toolbar, clicking Cookie-Editor, then click the
Export icon (which has a page with an arrow going out of it).
Click on JSON.

Then, in the debby folder, make a new file called cookies.json
and paste it in. (It is automatically copied to your clipboard.)

Then you can run the whole notebook.
